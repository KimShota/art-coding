# 🎨 art-coding

A collection of creative **Computer Vision & Machine Learning** projects that bring art and anime to life — built for everyone to explore, learn, and have fun with.

## 📁 Projects

### 🥊 [luffy_rubber_arms](./luffy_rubber_arms)
Real-time pose detection that visualizes Luffy's iconic rubber arm stretching effect using computer vision.

### ⚔️ [jujutsu_kaisen](./jujutsu_kaisen)
A computer vision project inspired by Jujutsu Kaisen, bringing cursed energy effects to life through code.

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe
- HTML / CSS / JavaScript

## 🚀 Getting Started
Each project folder contains its own files and instructions. Clone the repo and explore:

```bash
git clone https://github.com/KimShota/art-coding.git
cd art-coding
```
